== Description ==

Add reCAPTCHA anti-spam checking to Easy Digital Downloads registration and login forms


== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `reCAPTCHA for Easy Digital Downloads` from Plugins page

== Configuration ==

1. Register your domain on Google <a href="https://www.google.com/recaptcha/intro/index.html">reCAPTCHA service</a>;
2. Configure the options in Downloads -> Settings -> reCAPTCHA;
